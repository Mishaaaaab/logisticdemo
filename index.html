How the website works

  1. The client creates a request.
  2. The system generates a tracking number.
  3. The driver accepts a suitable shipment.
  4. The status can be checked online.

Client

Creates a request with route, weight, volume, and cargo description.

Driver

Sets vehicle capacity and chooses suitable orders.

Tracking

Each request receives a unique number for status tracking.
